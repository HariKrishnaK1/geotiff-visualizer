NDVI Science

NDVI = NIR - RED/NIR + RED

NIR represents Near-Infrared reflectance (high in healthy leaves).

RED represents Red light absorption (absorbed by chlorophyll).

Values closer to +1.0 represent dense, healthy vegetation, while negative values represent water, clouds, or barren lands.

NRSC GeoTIFF Viewer - Technical Guide

1. Platform Purpose

This decision-support system is an interactive prototype designed to demonstrate client-side web rendering of geographic raster datasets (GeoTIFF) and perform real-time vegetative analytics (NDVI).

2. How to Operate

  • Upload a GeoTIFF: Drag and drop any .tif or .tiff file that is georeferenced (e.g. WGS84 EPSG:4326 or UTM coordinate systems) into the upload zone.
  • Alternative Demo: Click the "Load Demo Scene" button to instantly render a synthetic multi-spectral (NIR, Red, Green) crop-plain scene centered in Hyderabad, India.
  • Run Analysis: Click the green "Analyze Dataset" button to zoom directly to the raster bounds, project the layer, and enable the live Pixel Inspector.
  • Interact with Map: Click anywhere on the rendered overlay area. The pixel inspector will calculate the exact coordinate position, the active spectral band values, and determine the vegetative index.
  • Compare Visualization Modes: Toggle between the color composites (NDVI Heatmap, False-Color Infrared, True-Color, or Grayscale) in the Overlay Controls to examine features.

3. Under the Hood

The client side runs georaster.js to decode metadata, geotagged matrices, and image tags of raw binary TIFF arrays directly in the browser's main thread. It then uses georaster-layer-for-leaflet.js to write dynamic color calculations onto an HTML Canvas overlaid on top of a web mapping layout powered by Leaflet.

System Specs: Offline Capability • Leaflet v1.9.4 • Georaster v1.6.0 • WebGL Canvas Accelerator Enabled
